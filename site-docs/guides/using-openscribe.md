# Using OpenScribe

## Daily workflow

1. Press the start or stop hotkey.
2. Speak and stop recording.
3. Review raw and polished text in Live.
4. Use copy or paste hotkeys as needed.

## Key UI areas

- **Live** for current run output and reruns.
- **History** for prior sessions and replay.
- **Stats** for aggregate and recent metrics.

## Common actions

- Re-transcribe a session with alternate provider or model.
- Re-polish a session with alternate provider or model.
- Reveal a session folder in Finder from History.
